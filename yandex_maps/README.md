# Анализ динамики метрик Яндекс Карт

## Цель проекта:
Построить масштабируемую аналитическую систему для анализа исторических метрик Яндекс Карт:
- выявление трендов и сезонности
- расчет годовой динамики (YoY)
- подготовка отчетных материалов для команд продаж рекламы

## Исходные данные
Исторические метрики активности пользователей Яндекс Карт  
Разрезы:
- индустрия → подгруппа → рубрика
- тип метрики
- помесячные значения за несколько лет

## Этап 1. Прототипирование и согласование логики (Excel + Power Query)

- Первичный расчет метрик
- Проверка бизнес-логики
- Согласование показателей с заказчиком
- Построение первого дашборда в DataLens

На этом этапе был собран первый дашборд в DataLens для обсуждения с заказчиком.

## Этап 2. Перенос в БД и масштабирование (PostgreSQL + Supabase)

**Слой исходных данных**

raw_maps — таблица с исходными данными  
Данные загружаются из CSV  
Исторические данные не пересчитываются  

**Слой преобразования данных**

v_maps_long — преобразование широкой таблицы в длинный формат  
v_maps_monthly — нормализация месяцев и лет  

**Аналитическая витрина данных**

v_maps_final — аналитическая витрина для BI
Содержит:
- помесячные значения,
- сравнение годов,
- YoY,
- min / avg / max,
- корректные сравнения только по общим месяцам

DataLens подключается напрямую к витрине

**Дашборд (Yandex DataLens)**  
Реализовано:
- Фильтры по индустриям, рубрикам и метрикам
- Сравнение значений по годам
- Вывод YoY и агрегированных показателей
- Использование дашборда как инструмента продаж

**Реализованные расчёты**  
Группировка данных по годам и месяцам  
Суммирование кумулятивных метрик  

Расчёт YoY:  
  ((текущий год – предыдущий год) / предыдущий год) * 100%  
Сводные статистики:
- минимальные значения
- средние значения
- максимальные значения
- Корректная логика сравнения неполных периодов

**Автоматизация обновлений**  
Добавление новых месяцев через CSV  
Данные дозагружаются без пересчёта истории  
Дашборды обновляются автоматически  
Инструмент используется без участия аналитика  

## Бизнес-результат:
- Сокращение времени подготовки аналитических материалов с ~2 недель до 4 часов.
- Рост количества аналитических срезов с 10 до 43.
- Переход от ежегодного обновления данных к ежеквартальному.
- Передача инструмента бизнес-командам для самостоятельной подготовки презентаций и анализа.
